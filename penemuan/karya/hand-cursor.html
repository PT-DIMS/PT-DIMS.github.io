Hand Cursor

Gerakkan kursor Windows hanya dengan lambaian tangan.

Selesai · v1 arsip #02
Cerita

Hand Cursor mengubah webcam biasa menjadi alat penunjuk. Lewat MediaPipe Hands, 21 titik sendi tangan dilacak real-time (via OpenCV), lalu diterjemahkan menjadi posisi kursor Windows melalui Win32 SetCursorPos — sadar-DPI, jadi presisi di layar resolusi apa pun.

Kosakata gesturnya lengkap: gerakkan jari manis untuk menggeser kursor, cubit jempol+telunjuk untuk klik kiri, cubit & tahan lalu geser untuk drag, cubit jempol+jari tengah untuk klik kanan, dan angkat telunjuk+tengah untuk scroll. Titik patokan sengaja dipasang di jari manis — jadi saat kamu mencubit untuk mengeklik, kursor tetap diam & klik mendarat tepat.

Yang bikin terasa halus bukan kebetulan. Koordinat dilewatkan One Euro Filter (Casiez dkk., 2012) — mulus saat tangan diam, responsif saat gerak cepat — dan kursor digerakkan oleh thread tersendiri di 144 Hz dengan easing, jadi tetap lembut walau FPS kamera rendah. Deteksi cubitan dinormalkan ke ukuran tangan (plus histeresis), sehingga konsisten di jarak berapa pun dari kamera.

Semua gerakan cukup di 'kotak aktif' kecil di tengah frame yang sudah memetakan seluruh layar. Dibungkus Python 3.11 terisolasi, dijaga single-instance mutex (biar dua instance tak berebut kursor), dan dijalankan cukup dengan satu klik RUN.bat atau lewat .exe-nya.

Fitur Unggulan

Patokan jari manis

Kursor mengikuti ujung jari manis — tetap stabil saat jari lain mencubit untuk klik.

5 gestur

Geser, klik kiri (cubit), klik kanan, drag (cubit-tahan), dan scroll dua jari.

Halus & anti-getar

One Euro Filter + thread glide 144 Hz bikin gerakan mulus walau FPS kamera rendah.

Konsisten segala jarak

Cubitan dinormalkan ke ukuran tangan + histeresis — klik andal dekat maupun jauh.

Kotak aktif

Area kecil di tengah frame memetakan seluruh layar; tak perlu jangkau pinggir.

Sekali klik

Jalan langsung lewat RUN.bat atau .exe, dijaga single-instance mutex.

Teknologi
Python 3.11 MediaPipe Hands OpenCV pyautogui ctypes / Win32 One Euro Filter Thread glide 144 Hz
