Agent AI Lokal

Otak dua-model yang berpikir 100% di mesin sendiri.

Aktif · eval 141/141 arsip #01
Cerita

Agent AI Lokal lahir dari satu prinsip: kecerdasan yang berdaulat. Seluruh inferensi berjalan lewat Ollama (server model di localhost:11434) di mesin sendiri — tak ada satu token pun yang bocor ke cloud. Di intinya, router.py bertindak sebagai otak: ia memilih model yang tepat untuk tiap tugas, lalu menjalankan agentic loop (tool-calling) dengan lapisan keandalan berupa retry, timeout adaptif, dan circuit-breaker.

Otaknya bukan satu model, melainkan sebuah tim model spesialis yang di-orkestrasi: qwen3.6 untuk obrolan & penalaran umum, qwen3-coder untuk urusan kode, qwen2.5vl untuk penglihatan (membaca layar/gambar), dan nomic-embed-text untuk mengubah teks jadi vektor — dipakai memory jangka panjang (recall via cosine similarity) dan knowledge base RAG lokal. Router bisa dijalankan dalam beberapa mode kolaborasi: orchestrate, pipeline, parallel, dan review.

Untuk mata ke dunia luar, agen memakai SearXNG yang berjalan di dalam Docker (kontainer sendiri, port 8888, restart unless-stopped). Mesin pencarinya di-hardening: reformulasi query, multi-query, lalu sintesis jawaban — dengan fallback otomatis ke Wikipedia bila perlu. Docker juga dipakai sebagai backend eksekusi disposable: run_shell, run_python, dan run_file bisa dijalankan di kontainer sekali-pakai (jaringan none secara default) — bisa debian-slim, python-slim, sampai kali-rolling untuk perkakas red-team.

Di atas semua itu berdiri 39 tool nyata (file, web, sistem, proses, media) yang dijaga safety gate + audit, serta keamanan berlapis Fase 1–4 + T2: aturan main (RoE) dengan scope engagement, mode blue-team, penjaga anti-injeksi, kontrol egress, hingga pelaporan otomatis. Semua aksi tercatat lewat observability (span/event → trace.jsonl, dashboard monitor, audit tool berisiko).

Ia bahkan punya wajah: J.A.R.V.I.S — HUD WebGL yang dibungkus pywebview, lengkap dengan suara (STT faster-whisper + TTS Edge-TTS) dan penglihatan. Hasil akhirnya: asisten operasional keamanan yang bisa dipercaya untuk latihan red & blue team — lulus 141 dari 141 eval regresi, dan tersimpan rapi di repo privat.

Fitur Unggulan

Tim model spesialis

qwen3.6 (chat), qwen3-coder (kode), qwen2.5vl (vision), nomic-embed (vektor) di-orkestrasi satu router.

Agentic loop + 39 tool

Tool-calling nyata (file/web/sistem/proses) dengan safety gate, audit, retry & circuit-breaker.

SearXNG + Docker

Search lokal ter-hardening (reformulasi, multi-query, sintesis) plus eksekusi kode di kontainer disposable.

Memory & RAG lokal

Ingatan jangka panjang + knowledge base, recall via embedding nomic-embed-text.

Keamanan berlapis

RoE + scope, blue-mode, anti-injeksi, dan kontrol egress bawaan (Fase 1–4 + T2).

100% lokal

Tanpa cloud — semua inferensi & data tetap di dalam mesin sendiri.
